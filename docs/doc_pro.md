أريد منك إنشاء **ملف Microsoft Word (.docx)** منسق ومرتب بشكل جميل (عناوين، فهرس، جداول، أيقونات بسيطة عند الحاجة، ألوان هادئة، محاذاة RTL للعربية) بعنوان:

**"تقرير مشروع روبوت الجامعة التقني (Voice + Chatbot)"**

## 1) معلومات المشروع (استخدمها كما هي)
- اسم المشروع المقترح: **Bot_IT – روبوت الجامعة التقني**
- فكرة المشروع: روبوت دردشة صوتي/نصي داخل الجامعة يجيب عن **الأسئلة التقنية والتكنولوجية** (برمجة، شبكات، أنظمة تشغيل، قواعد بيانات، ذكاء اصطناعي…)، مع رفض لطيف للأسئلة غير التقنية وتوجيه المستخدم لأسئلة ضمن النطاق.
- بيئة التشغيل: يعمل محلياً على جهاز الطالب (localhost)، ويمكن تطويره لاحقاً للنشر.
- واجهة عربية RTL.
- طريقة الإدخال الصوتي: **Web Speech API (SpeechRecognition)** في المتصفح لتحويل الكلام إلى نص.
- طريقة الرد: يولّد رد **نصي** عبر نموذج Gemini/Vertex AI، ثم يتم تحويل النص إلى **صوت** داخل المتصفح عبر **Web Speech API (SpeechSynthesis)**.
- الاتصال بين الواجهة والسيرفر: **WebSocket**.
- الواجهة الخلفية: Node.js + Express + ws.
- هيكل المشروع:
  - `backend/` (config, server, handlers)
  - `frontend/` (index.html, styles.css, app.js)
- ميزة “كلمة تنبيه”: الروبوت يرد فقط إذا بدأ السؤال بكلمة **"روبوت"** (مثال: "روبوت ما هي بايثون؟").

## 2) المطلوب في ملف Word (التنسيق والمحتوى)
### A) تنسيق وإخراج احترافي
- اتجاه الصفحة: RTL
- خط عربي مناسب (مثل: Cairo أو Arial) مع أحجام متدرجة
- صفحة غلاف (Cover Page) تحتوي:
  - اسم الجامعة (اترك خانة قابلة للتعبئة)
  - اسم المشروع
  - أسماء الفريق (خانات فارغة)
  - المشرف (خانة فارغة)
  - التاريخ
- فهرس تلقائي (Table of Contents) يعتمد على Styles للعناوين
- ترقيم صفحات
- استخدام جداول لتلخيص التقنيات والمتطلبات
- مربعات “ملاحظات/تنبيهات” قصيرة (Callouts) عند الحاجة
